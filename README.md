# InfnetFood

Aplicativo de delivery de alimentos desenvolvido em **React Native** com **Expo**.  
O app permite navegar por categorias de produtos, adicionar itens ao carrinho, realizar checkout com notificações simuladas, gerenciar perfil e pedidos, além de alternar entre tema claro e escuro.

---

## Funcionalidades

- Login mockado (qualquer e-mail e senha funcionam)  
- Listagem de categorias de refeições  
- Tela de produtos e detalhes de produtos  
- Carrinho de compras com alteração de quantidade e remoção de itens  
- Checkout com validação de endereço e método de pagamento  
- Notificações simuladas para pedido confirmado  
- Tela de perfil com informações do usuário  
- Tela de pedidos com status de cada pedido  
- Visualização de mapa com restaurantes  
- Tema claro e escuro (não aplicado ao login)
